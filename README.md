# TicTacToe game in PHP

Tic tac toe game made with PHP 8.1.6

# Installation
 ## Requirements
    * PHP 8.1.6
    * Composer
 ## Run
 Install the composer inside the project directory using:
      
      * composer install
 Open Powershell inside the project directory and execute the follwing command:
 
      * php main.php
